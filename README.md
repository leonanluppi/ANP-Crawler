# ANP-CRAWLER

#### How to install
 - Go to api directory
 - Make sure that you have MongoDB installed.
```sh
$ npm install
```
 - Go to web directory
```sh
$ npm install
$ bower install
```
#### How to run
 - Go to api directory
```sh
$ node index.js
```
 - Go to web directory
```sh
$ sudo gulp connect
```

NOTE: If you want to build this production. Just run: 
```sh
$ sudo gulp
```
